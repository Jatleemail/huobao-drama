# 剧本改写指南

## 改写原则

1. **保留核心情节**：不改变主线故事和角色关系
2. **增强画面感**：将叙述性文字转化为可视化的场景描写
3. **对话驱动**：用对白推动情节，减少旁白
4. **节奏把控**：每场戏控制在 30-60 秒，适合短视频
5. **不写镜头语言**：不涉及景别、角度、运镜，这些属于分镜拆解步骤

## 格式化剧本格式

```
## S01 | 内景 · 咖啡厅 | 黄昏

黄昏的光线透过落地窗洒进咖啡厅，吧台上咖啡杯热气升腾。

小明独自坐在角落卡座，低头看手机，神情有些焦虑。

门铃响起，小红推门而入。她看到小明，微笑着走过去。

小红：（微笑）等很久了吗？
小明：（抬头）还好，刚到。
```

### 格式规则

- `## S编号 | 内景/外景 · 地点 | 时间段` — 场景头
- 动作描写自然段 — 不包含任何镜头语言
- `角色名：（状态/表情）台词内容` — 对白格式

### 内容量参考

格式化剧本相比原始内容增加约 20-30%，主要增量是场景头标记和对白格式化，不是扩写。

### 对话质量规范

- 单句对白不超过 30 字，长对白适当分段
- 每句对白必须标注表情/状态，如：（愤怒）、（苦笑）、（压低声音）
- 避免信息型对白（用对白解释剧情），优先用动作和画面展现
- 角色间对话保持"给予-反应"节奏：A 抛出信息 → B 做出反应 → A 调整策略

## 改写技巧

- 心理描写可转化为角色表情/动作或画外音
- 长段叙述拆分为多个短场景
- 确保每个场景有明确的情绪转折点
- 保持角色语言风格一致性
- 场景编号连续递增（S01, S02, S03...）
- 时间段要具体（黄昏、深夜、清晨），不要笼统写"白天"

## 异常场景处理

- **原文缺少对白**：根据角色性格和上下文合理补充，保持语言风格一致
- **原文段落过长**：识别情绪转折点或场景切换点，拆分为 2-3 个短场景
- **内容不适合影视化**：纯说明性文字、元叙述或过于抽象的心理描写 → 标注原因并跳过，不强行改写
- **角色关系复杂**：保持所有互动关系不变，可在场景头添加简短的角色出场提示
- **角色对话冲突**：若多人同时说话，标记为交叉对话并用 / 分隔
  示例：`小明：/ 你听我说——/ 小红：/ 我不听！/`
