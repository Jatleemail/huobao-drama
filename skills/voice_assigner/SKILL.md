# 音色分配指南

## 分配原则

1. **性别匹配**：男性角色用男声，女性角色用女声
2. **年龄匹配**：少年/青年/中年/老年对应不同音色
3. **性格匹配**：
   - 活泼开朗 → 明亮有活力的音色
   - 沉稳内敛 → 低沉稳重的音色
   - 温柔体贴 → 柔和甜美的音色
   - 威严霸气 → 浑厚有力的音色
   - 阴险狡猾 → 尖锐或低沉带沙哑
   - 天真单纯 → 清脆明亮，偏童声
4. **角色定位**：主角用辨识度高的音色，重要配角与主角形成对比，龙套用中性音色

| 角色类型 | 推荐音色特征 | 建议方向 | 备注 |
|---------|------------|---------|------|
| 青年男主（善良型） | 明亮、温暖、中音 | 青涩青年类 | 辨识度高 |

| 豪爽型角色 | 浑厚、洪亮、语速快 | 真诚青年类 | 体现直率性格 |
| 少女/萝莉 | 清脆、甜美、偏高 | 少女音色类 | 注意年龄匹配 |
| 成熟女性 | 柔和、稳重、中音 | 御姐音色类 | |
| 反派（阴沉型） | 沙哑、低沉、语速慢 | 霸道类 | 不怒自威 |

## 冲突处理

当多个角色最适合同一音色时：
1. 主角优先获得
2. 其余角色选择次优匹配（音色库中相近但不完全相同的音色）
3. 在分配理由中标注冲突和替代决策

## 多角色同场场景协调

同一场景中 ≥3 个角色有对话时，按以下策略确保区分度：
1. 先分配主角（最独特音色）
2. 配角选择不同音高区间的音色
3. 同性别配角选择不同音色特质（明亮 vs 沉稳、柔和 vs 有力）
4. 龙套使用中性音色，不抢主角辨识度

### 协调示例（三和尚同场）

| 角色 | 音色 | 特征 |
|------|------|------|
| 小和尚 | 青涩青年音色 | 年轻、明亮 |
| 瘦和尚 | 精英青年音色 | 精明、中等 |
| 胖和尚 | 真诚青年 | 浑厚、直率 |

→ 三者音高/特质均有差异，听感不会混淆 ✓

## 区分度原则

同一场景中有对话互动的两个角色，音色应有足够的听觉区分度：
- 音高差异（高 vs 低）
- 音色特质差异（明亮 vs 沉稳、柔和 vs 有力）
- 避免两个角色使用过于相似的音色导致观众混淆
